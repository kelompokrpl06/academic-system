# 📄 Project Description  
Academic System adalah sistem informasi akademik berbasis web yang dirancang untuk membantu mahasiswa dalam mengelola kegiatan perkuliahan secara lebih mudah, cepat, dan terpusat. Sistem ini mencakup fitur seperti pemilihan KRS, evaluasi dosen (EDOM), bimbingan akademik, pengajuan revisi nilai, dan pengecekan tagihan kuliah.

Sistem ini dibuat untuk menggantikan proses manual menjadi digital agar lebih efisien, terstruktur, dan mudah diakses.

---

# 🎯 PROJECT OBJECTIVES
- Mempermudah mahasiswa dalam proses akademik  
- Mengurangi penggunaan proses manual di kampus  
- Mengintegrasikan layanan akademik dalam satu sistem  
- Meningkatkan efisiensi pengelolaan data akademik  
- Memberikan pengalaman pengguna yang lebih mudah dan modern  

---

# ⚙️ FITUR SISTEM
- 📌 Pemilihan KRS (Kartu Rencana Studi)  
- 📊 Evaluasi Dosen (EDOM)  
- 👨‍🏫 Bimbingan Akademik  
- ✏️ Pengajuan Revisi Nilai
